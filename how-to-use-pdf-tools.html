Practical guide

How to use PDF tools safely and get reliable results

PDF work is simple when the task is clear. Start with a copy of the original file, use the tool that matches the task, and open the finished file before sharing it.

Merge PDFs

Use Merge PDF when several documents need to become one file. Add every source document, check their order, then create the result. For reports and applications, review the first, last, and transition pages after merging to make sure no page was skipped or duplicated.

Split or extract pages

Split PDF creates a new file from a page range. Extract Pages is the better choice when you need selected pages rather than one continuous section. Check the source document's page numbers carefully; printed page numbers and PDF page indexes can differ when a cover sheet is present.

Rotate and compress

Rotate PDF is useful for scanned pages that are sideways. Compression can make email attachments easier to send, but it may reduce image quality. Keep the original if the PDF contains detailed diagrams, small text, or photos that must be printed.

Convert files

PDF-to-image conversion is useful when a page must be placed in a presentation or shared as a picture. Image-to-PDF is useful for scans and photos; arrange images in the intended order before exporting. Word and PDF conversions can change complex layouts, fonts, forms, or comments, so always inspect the final document.

Protect documents thoughtfully

Password protection helps prevent casual access, but it cannot recover a forgotten password. Choose a strong password and share it separately from the file. Encryption does not replace a secure backup or permission to distribute the document.

Privacy checklist

  • Use a current browser and close the tab when you finish.
  • Keep a copy of the original document before processing.
  • Review outputs before sending them to someone else.
  • Do not upload or email sensitive documents unless you understand the recipient and service involved.
